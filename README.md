One minor but enduring annoyance in my legal practice is how cumbersome it is to pull up the "official" versions of the New Hampshire RSAs. The State website (https://gc.nh.gov/rsa/search/default.aspx), in my opinion, has a UI that is inelegant and difficult to use. To make matters worse, when you do a Google search for a particular RSA, the State website doesn't even show up much of the time. The top results are usually aggregator websites like Justia, which aren't always up to date. For my peace of mind, I end up having to either search in Lexis or manually browse the State website for the statute I'm looking for.

I therefore created (with the help of ChatGPT!) a little online tool that I think is a vast improvement on the existing options to navigate the RSAs. It is available at: https://izdr1.github.io/NH-RSA/. I hope it is of use!
